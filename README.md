# class-components
Week 3: Git contribution to class repo

1. Fork the iHub class components repo (link in Google Drive).
2. Create a branch named feature/yourname-component. e.g. 'feature/Kat-card'
3. Build one small HTML/CSS UI component in /components/yourname/ (card, nav, footer, hero, etc.).
4. Open a pull request to the main class repo with a CLEAR DESCRIPTION: what you BUILT and what CSS technique you used.
5. Leave a constructive review comment on at least one other trainee's PR.

DELIVERABLE:
Submit the link to your open PR and the link to your review comment on a peer's PR.
